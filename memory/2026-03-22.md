# Session Notes: 2026-03-22

## Lessons Learned

### PowerShell & Executables
- `gh` command not in PATH initially, but full path works: `C:\Program Files\GitHub CLI\gh.exe`
- PowerShell quoting is tricky with executables; use `cmd /c "path.exe" args` for clean execution
- Quotes in paths break PowerShell parsing—use `cmd /c` wrapper instead

### Weather APIs
- `wttr.in` routes through local proxy that can be down
- Open-Meteo API is a reliable fallback: `https://api.open-meteo.com/v1/forecast?latitude=40.79&longitude=-73.95&current_weather=true`
- Use `Invoke-WebRequest` with `-UseBasicParsing` in PowerShell

### GitHub CLI
- `delete_repo` requires explicit scope: `gh auth refresh -h github.com -s delete_repo`
- Can create repos with `repo create` but deletion needs admin rights/scopes

### OpenClaw & Tool Calling
- `exec-approvals.json` must exist for `exec` tool to be available
- Fresh sessions prevent poisoned conversation history from confusing the model
